ABOUT MEETJ

Welcome to meetJ. We hope that as you use this website you will meet Jesus and learn how much He cares about you and wants to help you know Him.

How to use the website

Each new day choosing the next day's readings Read through the Bible verses and notes and then, if you are a registered user you can mark the day as "read". This will enable you to go to the list of days and see which ones you have finished and which days you still need to complete

How to pray

The main advantage of being a registered user of meetJ is that after reading the Bible and notes for the day you can mark the day as “read". This will enable you to go to the list of days and see which ones you have finished and which days you still need to complete. We will also send you a reminder email as a prompt to login and read the next day’s reading.
